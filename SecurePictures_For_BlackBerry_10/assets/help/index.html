Secure Pictures is a secure camera shots and voice recording protection solution for BlackBerryÂ® 10 devices.

1. SETUP

A setup wizard guides the user through initialization of the application.

The user must enter a password that will be used as a security key for the AES-256 symmetric encryption.

The password must contain at least 6 characters but it is advised to contain at least 32 characters (256 bits) to achieve the highest protection.

Password should contain combination of upper case letters, lower case letters, numbers and special characters.

2. TAKING A PICTURE, SHOOTING A VIDEO OR RECORDING A SOUND

Tap on a Camera, Video Camera or Dictaphone action to invoke smartphone's camera for taking the picture or shooting the video, or dictaphone to record a sound.

Camera shot or voice record is automatically encrypted when the action is finished. Secure pictures have .sjpg file extension, secure videos have .smp4 file extension and secure voice records have .sm4a file extension.

3. VIEWING SECURE MEDIA FILE

Browse through folders and tap on a file to invoke a picture viewer, a music player or a video player. User must use the password to unlock the file.

You can also invoke the viewer from external applications by tapping on secure files from email client or file manager.

By tapping and holding on file, a context menu is activated where the file can be renamed and deleted.

4. SETTINGS

In settings page, user can:

  • Turn on or off multiple camera shooting,
  • Turn on or off forgetting the password when minimizing the application,
  • Turn on or off forgetting the password when leaving the viewer or the player, and
  • Change the password.
