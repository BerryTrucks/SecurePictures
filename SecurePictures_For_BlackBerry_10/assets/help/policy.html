The Secure Pictures application does not transmit any personal data via Internet or through any other means of electronic communication.

Required permissions:

  • The Camera and Microphone permissions are required for the application to be able to take a picture, shoot a video or record a sound.
  • The Shared Files permission is required for accessing pictures and videos.
  • The Run as Active Frame permission is required for the application to run when minimized.

If you have any concerns regarding privacy and security, please don't hesitate to contact us at info@pronic.si.
